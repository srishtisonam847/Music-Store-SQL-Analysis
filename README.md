
# Music Store SQL Analysis

## Objective

Analyze customer behavior and sales performance using SQL.

## Database

Music Store Database

## Skills Used

- Joins
- Aggregations
- Subqueries
- CTEs
- Group By
- Order By

## Business Questions

1. Senior most employee
2. Country with most invoices
3. Best customer
4. Rock music listeners
5. Top selling artist
6. Revenue by city

## Tools Used

- PostgreSQL
- DBeaver
